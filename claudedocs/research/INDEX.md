# Research Documents Index

This directory contains topic-specific research findings, validation studies, and implementation analyses organized by research area.

## Directory Structure

- **[strikethrough/](strikethrough/)** - Strikethrough and *sous rature* detection research
- **[pdf-processing/](pdf-processing/)** - PDF processing and formatting research
- **[metadata/](metadata/)** - Metadata extraction and enhancement research

## Research Areas

### Strikethrough Detection
Computer vision approaches, validation studies, and implementation strategies for detecting strikethrough text in academic PDFs, including Derrida's *sous rature* (under erasure).

### PDF Processing
Advanced PDF processing techniques including garbled text detection, formatting group merger architecture, and quality analysis frameworks.

### Metadata Extraction
Enhanced metadata extraction strategies, publisher detection, and metadata validation approaches.
